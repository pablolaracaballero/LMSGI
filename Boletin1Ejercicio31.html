Intruducción

MySQL es un sistema de gestión de bases de datos relacional (RDBMS) de código abierto que utiliza el lenguaje SQL (Structured Query Language) para gestionar y manipular bases de datos. Fue desarrollado por MySQL AB, una empresa sueca, y posteriormente adquirido por Sun Microsystems, que luego fue adquirida por Oracle Corporation.
