HTML

HTML es el lenguaje de marcas para la creación de páginas web.

HTML significa HyperText Markup Language (Lenguaje de Marcado de Hipertexto).

Historia de HTML

Fue creado por Tim Berners-Lee en 1991 y ha sido la piedra angular de la World Wide Web desde entonces.

Historia de HTML

HTML se utiliza para estructurar el contenido en la web, permitiendo a los desarrolladores definir elementos como encabezados, párrafos, enlaces, imágenes, listas y otros componentes esenciales de una página web.
